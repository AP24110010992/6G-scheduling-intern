# Nash Equilibrium

## Definition

Nash Equilibrium is a game theory concept where no player can improve their result by changing their strategy while the other player keeps their strategy unchanged.

## Example

Attacker Strategies:

* Launch Attack
* Stay Idle

Defender Strategies:

* Enable Defense
* Disable Defense

When both attacker and defender choose the most stable strategy and neither gains advantage by changing alone, the system reaches Nash Equilibrium.

## Application

Nash Equilibrium is commonly used in cybersecurity, wireless communication, and network resource management to study stable decision-making between attackers and defenders.

# Stackelberg Game Theory

## Definition

Stackelberg Game is a leader-follower strategic model used in game theory.

## Roles

Leader:

* Defender or Network Controller

Follower:

* Attacker or Secondary User

## Working Process

1. The defender selects a protection strategy first.
2. The attacker observes the defender’s action.
3. The attacker responds based on the chosen defense strategy.

## Application

Stackelberg games are widely used in IoT security, cyber defense systems, intelligent scheduling, and dynamic resource allocation because they help defenders act proactively against attacks.

# Modbus Dataset Summary

## Dataset Name

Modbus Network Traffic Dataset

## Purpose

The dataset is used for Industrial IoT security analysis and attack detection.

## Important Features

* Source IP Address
* Destination IP Address
* Communication Protocol
* Packet Size
* Timestamp Information
* Function Codes

## Common Attack Types

* Denial of Service (DoS)
* Network Scanning
* Malicious Injection Attacks

## Usage

The dataset can be used to train machine learning models for:

* Intrusion Detection
* Threat Classification
* Network Monitoring
* Abnormal Traffic Detection
* Predictive Cybersecurity Analysis
