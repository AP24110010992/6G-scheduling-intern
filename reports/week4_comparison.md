# Detection Methods Comparison

| Method | Technique | Accuracy |
|---------|-----------|----------|
| Rule-Based | Risk Threshold | Baseline |
| Random Forest | Machine Learning | 93.17% |
| MLP | Deep Learning | 77.17% |

## Summary

Three attack detection methods were implemented during the internship.

The rule-based defender uses manually defined rules.

The Random Forest model learns attack patterns from historical data and achieved the highest accuracy.

The MLP model uses a neural network to classify network traffic. Although its accuracy was lower than Random Forest in this experiment, it demonstrates the use of deep learning for intrusion detection.
