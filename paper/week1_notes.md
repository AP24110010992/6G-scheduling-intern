Week 1 Learning Summary

During the first week of the internship, basic concepts related to event-driven scheduling and 6G network simulation were studied.

Python fundamentals were learned to support simulation and network programming tasks. Topics such as functions, loops, conditions, and data structures were practiced.

NetworkX was explored for creating and visualizing communication network topologies. Basic node and edge creation methods were understood.

SimPy was introduced for event-driven simulation. Basic process scheduling and time-based event execution concepts were learned.

Game theory concepts including Nash Equilibrium and Stackelberg Game Theory were studied to understand intelligent resource allocation and scheduling decisions in future 6G networks.

The Modbus dataset was reviewed to understand industrial network communication and attack detection datasets used in IoT environments.

A basic understanding was developed on how graph theory, simulation, and intelligent scheduling can be combined for future research in 6G systems.

Future work includes attack simulation, IoT security modeling, network monitoring, and large-scale network topology creation.
